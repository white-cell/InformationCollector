# InformationCollector
* 针对域名目标收集子域名
* 获取子域名指纹和搜索引擎部分结果
* 通过证书信息在censys挖掘源ip
* 通过子域名判断是否使用cdn挖掘源ip

# tips
* pip install bs4
* 42、43行配置censys的key
* 有的网站api需要翻墙，37行可自行配置代理
* 还有很多很棒的接口，但是都会有次数限制，就像censys一个月只能100次

# 输出
* 子域名相关指纹和爬虫信息输出到html文件
* 子域名输出到txt
* 源ip输出到txt
